self-hosting node.js React
Minimal React application to display single page of posts.

Using shared model, read *.json that defines page of posts and display its content: 
    https://github.com/mstandio/JSProjects/tree/main/260306-blog-viewer
So for now I need to have Post Component that renders single Post, then Page Component that maps array of Posts from *.json to list of rendered Posts. On top of that I need to read a single *.json file deserialize it to shared Model type and pass it along to Page component.

All this vibe coded "Jesus-take-the-wheel" style which did surprisingly work. Module setup worked nicely in monorepo format, I even got some passing tests out of this.
